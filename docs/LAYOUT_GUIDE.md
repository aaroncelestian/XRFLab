## Color Scheme

### Main Colors
- **Background**: `#f5f5f5` (light gray)
- **Panels**: `#ffffff` (white)
- **Borders**: `#cccccc` (medium gray)
- **Text**: `#333333` (dark gray)

### Accent Colors
- **Primary**: `#2196F3` (blue) - Focus, selection
- **Success**: `#4CAF50` (green) - Fit button, positive
- **Info**: `#2196F3` (blue) - Export button
- **Warning**: `#FF9800` (orange) - Warnings
- **Error**: `#F44336` (red) - Errors

### Plot Colors
- **Measured**: `#0000FF` (blue) - Main spectrum
- **Fitted**: `#FF0000` (red) - Fitted spectrum
- **Background**: `#00FF00` (green) - Background
- **Residuals**: `#0000FF` (blue) - Residual points
- **Grid**: `#E0E0E0` (light gray, 30% alpha)

## Interactive Features

### Spectrum Plot
- **Mouse Move**: Crosshair follows cursor
- **Mouse Wheel**: Zoom in/out
- **Click + Drag**: Pan view
- **Right Click**: Context menu (zoom, export)
- **Double Click**: Auto-range

### Element Tree
- **Click Checkbox**: Select/deselect element
- **Type in Search**: Filter elements
- **Click Category**: Expand/collapse
- **Select All**: Check all visible elements
- **Clear All**: Uncheck all elements

### Results Table
- **Click Row**: Highlight element
- **Double Click**: Show element details
- **Right Click**: Context menu (copy, export)

## Keyboard Shortcuts

### File Operations
- `Ctrl+O` - Open spectrum
- `Ctrl+S` - Save project
- `Ctrl+W` - Close spectrum
- `Ctrl+Q` - Quit application

### Analysis
- `Ctrl+F` - Fit spectrum
- `Ctrl+Q` - Quantification
- `Ctrl+B` - Background settings

### View
- `Ctrl+L` - Toggle log scale
- `Ctrl+G` - Toggle grid
- `Ctrl+T` - Toggle theme

### Navigation
- `Ctrl++` - Zoom in
- `Ctrl+-` - Zoom out
- `Ctrl+0` - Reset zoom
- `F11` - Fullscreen

## Responsive Behavior

### Window Resize
- Panels maintain proportions (30:50:20)
- Splitters allow manual adjustment
- Minimum window size: 1000x600 px
- Layout adapts to available space

### Panel Collapse
- Double-click splitter to collapse panel
- Double-click again to restore
- Collapsed state saved in settings
